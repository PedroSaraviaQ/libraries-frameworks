Grid System

A responsive systems that can layout and align content. To start with, everything must be inside a "container". Then, we use "row" and "col" classes.

You can just use the "col" class for every column to be the same width.


    <div class="container">
      <div class="row">
        <div class="col">
          <div>abc</div>
        </div>
        <div class="col">
          <div>123</div>
        </div>
        <div class="col">
          <div>xyz</div>
        </div>
      </div>
    </div>
abc
123
xyz

You can also add numbers from "1 to 12", or even "auto" to resize to its content. If we exceed the 12 grid, the rest will just go to the next row.


    ...
    <div class="col">...</div>
    <div class="col-6">... </div>
    <div class="col">...</div>
    ...
abc
123
xyz

    ...
    <div class="col-2">...</div>
    <div class="col-7">... </div>
    <div class="col-auto">...</div>
    ...
abc
123
xyz
