Buttons

They should have the basic "btn" class. Additionally, you should add other classes with the prefix "btn-" for applying styles.

Anchor and input elements can also have these classes applied.

Colors

They match the basic colors, and the background changes a little when hovering.


  class="btn btn-primary"

Sizes

You can also have larger (lg) or smaller (sm) buttons.


  class="btn btn-sm"
  class="btn btn-lg"

Outline

They have a transparent background that changes to the applied color when hovering. The class should also have the "outline-" prefix.


  class="btn btn-outline-danger"

Groups

Buttons normally have space between them. To stack them together (and remove the border-radius between them), nest them inside an element with either "group" or "group-vertical". 


  <div class="btn-group">

  ...buttons

  </div>


  <div class="btn-group-vertical">

  ...more buttons

  </div>
