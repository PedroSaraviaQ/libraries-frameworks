Setting up Bootstrap

NPM

  1. Install Bootstrap inside the project.
    2. 
   npm i bootstrap
  2. In the HTML file, link the "bootstrap.min.css" file from the "node_modules/bootstrap/dist/css" folder.
    3. 
   <link rel="stylesheet" href="../bootstrap.min.css">
  3. Optionally, link the "bootstrap.bundle.min.js" file for dynamic components.
    4. 
   <script src="../bootstrap.bundle.min.js"></script>

Compiled versions

Download the compiled files for both CSS and JS and link them the same way as the previous one.


    <link rel="stylesheet" href="../bootstrap.min.css">
    <script src="../bootstrap.bundle.min.js"></script>

CDN

Link the compiled files the same way, but from their website. Additionally, both "link" and "script" tags are going to have the "integrity" (has to match the hash value to get executed) and "crossorigin" attributes for security measures.


   <_____ ... integrity="sha384-..." crossorigin="anonymous">
